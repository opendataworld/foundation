# OpenDataWorld Capability Map

OpenDataWorld provides data and analytics solutions as a service for everyone.

This capability map defines the reusable business and platform capabilities required to deliver that promise.

## Capability North Star

```text
Help everyone discover, understand, trust, govern, access, analyze, explain, decide, act, monitor, learn, and improve with data.
```

## Minimum Viable Capability Baseline

For a useful first platform release, OpenDataWorld should support:

1. Register data assets
2. Classify data assets
3. Describe datasets with metadata
4. Attach schema references
5. Assign owner and steward
6. Define access level
7. Publish dataset pages
8. Search and discover data assets
9. Record provenance
10. Track quality tier
11. Request access
12. Record audit events

## Strategic Capability Statement

OpenDataWorld is a service fabric for data and analytics.

It turns fragmented data into governed, trusted, reusable services that support analytics, insight, decision, action, and value for everyone.

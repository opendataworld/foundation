# OpenDataWorld Foundation

OpenDataWorld Foundation is the canonical foundation for an open, AI-native, cloud-native data intelligence ecosystem.

It defines the shared language, schemas, governance model, ontology, lifecycle, and operating principles for trustworthy data, knowledge, agents, and decision intelligence.

## North Star

Enable institutions, builders, researchers, governments, communities, and enterprises to make data findable, trustworthy, interoperable, reusable, governable, and actionable.

## Core Beliefs

- Everything is an asset.
- Everything has metadata.
- Everything has provenance.
- Everything is governed.
- Everything is traceable.
- Everything is versioned.
- Everything is observable.
- Everything should interoperate through open standards.
- Current state is a projection; history and provenance are first-class.
- AI assistance must be explainable, auditable, and human-governed.

## Scope

This repository defines the canonical foundation for:

- Data assets

- APIs

## Standards Alignment

OpenDataWorld Foundation should align with:

- schema.org
- JSON-LD
- DCAT
- FAIR data principles
- OpenAPI
- AsyncAPI
- OpenLineage
- OpenMetadata
- Open Policy Agent
- W3C DID
- W3C Verifiable Credentials
- CNCF cloud-native patterns
